# Configuración de APIs Satelitales Reales - Paso a Paso

## 🚀 Registro de APIs Gratuitas (5-10 minutos total)

### 1. NASA API (Más importante - Elimina error 429)
🔗 **Ir a:** https://api.nasa.gov/
📝 **Pasos:**
1. Hacer clic en "Get Started"
2. Llenar el formulario (nombre, email, descripción: "Educational satellite data analysis")
3. Copiar la API key que te envían al email
4. Límites: 1,000 requests/hora (vs 30 con DEMO_KEY)

### 2. N2YO - Tracking de Satélites en Tiempo Real
🔗 **Ir a:** https://www.n2yo.com/api/
📝 **Pasos:**
1. Crear cuenta gratuita
2. Ir a "API" en el menú
3. Obtener API key gratuita
4. Límites: 1,000 requests/día

### 3. OpenWeatherMap - Datos Meteorológicos Satelitales  
🔗 **Ir a:** https://openweathermap.org/api
📝 **Pasos:**
1. Crear cuenta gratuita
2. Ir a "API keys" en el dashboard
3. Copiar la API key por defecto
4. Límites: 1,000 calls/día
